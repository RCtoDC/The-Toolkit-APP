**Content:**

**2\. FINAL BEST PRACTICE**

**BEST PRACTICE 1**

**Multi-Method Benchmarking on a Single Object**

**Object:** _John the Baptist_  
**Institution:** National Gallery of Slovenia

**Context and Goal**

This best practice demonstrates how different digitization methods and software packages produce varying results when applied to the same original object. The goal was to understand the impact of technology choices on quality, usability, and reuse potential.

**Equipment and Software**

- Multiple recording devices (3D scanners, cameras, video)
- Multiple software packages
- Identical object and conditions

**Workflow Summary**

- Capture the same object using different technologies
- Process data using different software pipelines
- Produce six distinct metaworks
- Compare results across quality, accuracy, and reuse potential

**Result Characteristics**

- Variations in geometry accuracy
- Differences in texture quality
- Different file sizes and processing times

**Strengths**

- Evidence-based decision-making
- Excellent training and benchmarking tool
- Supports institutional learning

**Limitations**

- Resource-intensive
- Not required for routine digitization

**Why This Is a Best Practice**

This approach enables institutions to **benchmark methods before standardizing workflows**, reducing long-term risks and improving strategic planning.

**Best Practice Principle:**

_Benchmark before you standardize._

**BEST PRACTICE 2**

**Research-Grade Digitization for High-Value Objects**

**Object:** _Altar of the Holy Cross_  
**Institution:** Co-Cathedral of the Holy Cross, Križevci

**Context and Goal**

This best practice represents the digitization of a culturally significant object requiring high accuracy, completeness, and long-term reuse potential for research and conservation.

**Equipment and Software**

- Professional 3D scanner or high-end photogrammetry setup
- Controlled lighting environment
- Advanced processing software

**Workflow Summary**

- Detailed planning and object assessment
- High-resolution capture
- Careful processing and validation
- Comprehensive metadata documentation

**Result Characteristics**

- High geometric precision
- Detailed textures
- Long-term archival value

**Strengths**

- Suitable for research and conservation
- High documentation quality
- Reference-level digital twin

**Limitations**

- Higher cost
- Requires expertise and time

**Why This Is a Best Practice**

This workflow demonstrates when and how to invest in **maximum quality digitization** for objects of exceptional value.

**Best Practice Principle:**

_Invest more where cultural and scientific value is highest._

**BEST PRACTICE 3**

**Scalable Digitization for Broad Access**

**Object:** _St. Stephen, kneeling_  
**Institution:** National Gallery of Slovenia

**Context and Goal**

This best practice focuses on efficient digitization using accessible tools, enabling institutions to digitize larger parts of their collections with limited resources.

**Equipment and Software**

- Smartphone or mid-range camera
- Affordable photogrammetry software
- Minimal setup

**Workflow Summary**

- Fast object preparation
- Image capture with sufficient overlap
- Simplified processing
- Basic quality control

**Result Characteristics**

- Good visual quality
- Lower metric accuracy
- Lightweight files

**Strengths**

- Low cost
- Fast and repeatable
- Ideal for education and exhibitions

**Limitations**

- Not suitable for conservation analysis
- Lower precision

**Why This Is a Best Practice**

This workflow ensures **inclusivity and scalability**, especially for small and medium institutions.

**Best Practice Principle:**

_Good enough, done well, enables access._

Module 8: Best Practices presents three complementary digitization approaches derived from the RCtoDC Toolkit implementation. Rather than promoting a single optimal method, the selected examples demonstrate how cultural heritage institutions can align digitization workflows with object value, intended reuse, and available resources. The best practices support evidence-based decision-making, scalability, and sustainability in line with the objectives of the European Data Space for Cultural Heritage and the Europeana Impact Playbook.
